# Smart-complaint-management-system
✨ AI Resume Screener is an intelligent tool that automates hiring by analyzing resumes 📄 and matching them with job descriptions using AI 🤖 and NLP. Built with Python and scikit-learn, it extracts skills 🔍, calculates match scores 📊, ranks candidates 🏆, and suggests missing skills 📌, making recruitment faster ⚡, efficient, and unbiased.
